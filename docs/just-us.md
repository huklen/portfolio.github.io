# Just Us 
![Image](../static/img/justus.gif)

## Information
 A co-op multiplayer survival game developed entirely as a solo project.

## Tasks:
* Networking programming
* AI programming
* Gameplay mechanics
* Weather systems
* Animation programming
* Dedicated servers & listen servers
* Matchmaking and much more
